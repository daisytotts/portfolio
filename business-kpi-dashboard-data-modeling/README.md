# Business KPI Dashboard & Data Modeling

This project focuses on building a small business KPI dashboard using SQL and Power BI.

The goal is to show how raw business data can be cleaned, structured, modeled, and turned into useful business insights.

## Tools

* SQL
* Power BI
* Python
* Data modeling
* Business reporting

## Main Steps

* Data cleaning
* SQL table creation
* KPI calculation
* Data modeling
* Dashboard design
* Business insight summary

## Example KPIs

* Revenue
* Profit
* Customer performance
* Sales trends
* Product performance
* Regional performance

## Project Flow

Raw data → Python cleaning → SQL database → Power BI dashboard → Business insights
